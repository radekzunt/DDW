Phone numbers extraction
===

The extraction of (Czech) phone numbers from a web page. Following formats are supported:

<ul>
<li>+420 607 123 456</li>
<li>420 605 123 456</li>
<li>+420777123456</li>
<li>+420 777123456</li>
<li>604123456</li>
<li>776 123 456</li>
</ul>


Used technologies
--

<ul>
<li>Java</li>
<li>GATE Embedded</li>
<li>JAPE grammar</li>
</ul>
